# Portafolio de Ingeniería en Informática

Este repositorio reúne una colección de proyectos académicos desarrollados durante la carrera de Ingeniería en Informática. Incluye trabajos enfocados en programación, bases de datos, análisis de datos, inteligencia de datos, desarrollo de software e Internet de las Cosas (IoT).

## 📌 Objetivo

Consolidar y documentar proyectos realizados en distintas asignaturas, aplicando conocimientos de ingeniería de software, análisis de datos y desarrollo de sistemas, con el fin de demostrar competencias técnicas adquiridas durante la formación académica.

## 📁 Proyectos incluidos

### 💻 Desarrollo de Software

Sistema de gestión de minimarket (Java + MySQL)
API REST para minimarket (Spring Boot)

### 📊 Análisis de Datos y Machine Learning

Limpieza y análisis del dataset Spotify 2023
Comparación de modelos de regresión (Spotify 2023)
Análisis de regresión lineal (League of Legends: kills vs oro)

### 📡 Internet de las Cosas (IoT)

Sistema de alerta inteligente con ESP32 y MQTT (simulado en Wokwi)
Sistema IoT de detección de alertas por voz con Node-RED

## 🚀 Propósito

Este repositorio tiene fines académicos y de aprendizaje, pero también sirve como portafolio técnico para demostrar habilidades en desarrollo de software, análisis de datos e implementación de sistemas.
