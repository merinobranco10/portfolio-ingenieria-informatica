Comparación de modelos de regresión aplicados al dataset Spotify 2023, con el objetivo de predecir la variable streams. El proyecto evalúa y contrasta el desempeño de distintos enfoques de machine learning, incluyendo Árbol de Decisión, Random Forest y Regresión Ridge, utilizando métricas de evaluación y visualizaciones comparativas. Implementado y ejecutado en Google Colab con Python.
